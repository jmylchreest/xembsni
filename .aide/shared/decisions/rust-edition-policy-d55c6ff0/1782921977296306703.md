# rust-edition-policy

**Decision:** Always use the latest stable Rust edition; set rust-version (MSRV) in Cargo.toml

## Rationale

Each edition unlocks safety improvements and language features. The latest edition (currently 2024) adds critical safety defaults like unsafe_op_in_unsafe_fn warnings. MSRV communicates the minimum compiler version to downstream users.

## Details

Set edition in workspace Cargo.toml. Migrate with cargo fix --edition. Set rust-version to minimum supported compiler. CI: test on stable + MSRV.

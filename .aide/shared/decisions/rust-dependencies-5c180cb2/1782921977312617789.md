# rust-dependencies

**Decision:** Use workspace dependency inheritance; run cargo-deny for license compliance and vulnerability scanning

## Rationale

Workspace inheritance eliminates version duplication across crates. cargo-deny subsumes cargo-audit and adds license, ban, and source checks — critical for production software.

## Details

Define deps once in [workspace.dependencies]; members use dep.workspace = true. SemVer ranges ("1.2" not "=1.2.3"). Commit Cargo.lock for binaries. Configure deny.toml for advisories/licenses/bans/sources; run in CI per PR. Weekly cargo audit as backup. Use cargo-semver-checks before publishing libs.

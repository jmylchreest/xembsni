# rust-ci-coverage

**Decision:** Use cargo-llvm-cov for code coverage; upload to Codecov

## Rationale

cargo-llvm-cov uses LLVM source-based instrumentation for accurate, region-level coverage. It is cross-platform (Linux/macOS/Windows), unlike cargo-tarpaulin which is Linux-only and ptrace-based.

## Details

Run: cargo llvm-cov --workspace --lcov --output-path lcov.info. Upload: codecov/codecov-action@v4, file: lcov.info. Target 80%+. Run on main merges for trends; optionally on PRs for diff coverage.

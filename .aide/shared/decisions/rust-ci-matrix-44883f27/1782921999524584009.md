# rust-ci-matrix

**Decision:** Test on stable + MSRV; optionally nightly with continue-on-error; cross-platform matrix for binaries

## Rationale

Stable catches current issues. MSRV ensures downstream compatibility. Nightly previews upcoming breakage but should not block PRs. Cross-platform testing catches platform-specific bugs.

## Details

Matrix: rust: [stable, "1.XX" (MSRV)], os: [ubuntu-latest, macos-latest, windows-latest]. Nightly: continue-on-error: true. Set fail-fast: false. Cross-compile: houseabsolute/actions-rust-cross. Feature combos: cargo hack test --feature-powerset --depth 2.

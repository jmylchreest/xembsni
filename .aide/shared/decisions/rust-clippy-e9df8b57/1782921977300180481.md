# rust-clippy

**Decision:** Configure lints in Cargo.toml with workspace inheritance; enable pedantic and nursery with selective allows

## Rationale

Cargo.toml lint configuration (RFC 3389) is the modern approach — workspace inheritance ensures consistency across all crates. Pedantic catches real issues but has noisy lints that should be selectively allowed. Restriction group should never be enabled wholesale — cherry-pick specific lints.

## Details

[workspace.lints.clippy]: all/pedantic/nursery/cargo at warn, priority -1. Allow: cast_precision_loss, module_name_repetitions, must_use_candidate. Restriction cherry-picks: dbg_macro, undocumented_unsafe_blocks, unwrap_used. Members: [lints] workspace = true. CI: clippy --workspace --all-targets --all-features, RUSTFLAGS="-Dwarnings".

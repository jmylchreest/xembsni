# rust-release-automation

**Decision:** Use release-plz for version bumping and changelog; cargo-dist for binary packaging and GitHub Releases

## Rationale

release-plz automates version bumping from Conventional Commits, changelog generation via git-cliff, and crates.io publishing. cargo-dist handles cross-platform binary builds, installers, and GitHub Release uploads. They complement each other.

## Details

release-plz: push-to-main triggers release PR (version bump + changelog). Merge publishes to crates.io. cargo-dist: tag push triggers cross-platform binary builds + GitHub Release with installers (shell, PowerShell, Homebrew). Simpler alt: cargo-release for versioning + publishing in one tool.

# rust-testing

**Decision:** Multi-layered testing with cargo-nextest as runner; property testing with proptest; snapshot testing with insta

## Rationale

cargo-nextest is up to 3x faster than cargo test with per-test process isolation and flaky test detection. proptest catches edge cases manual tests miss. insta makes golden-file testing trivial with cargo insta review.

## Details

Layers: unit (#[cfg(test)]), property (proptest), snapshot (insta + cargo insta review), integration (tests/), doc tests. Use rstest for parameterized/fixtures, mockall for trait mocking. Target 80%+ coverage via cargo-llvm-cov. CI: cargo nextest run --workspace.

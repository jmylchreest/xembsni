# rust-ci-sanitizers

**Decision:** Run sanitizers in CI: ASan+UBSan together, TSan separately, optionally MSan

## Rationale

Sanitizers catch memory safety and concurrency bugs that tests alone miss. ASan and TSan are mutually exclusive and must run as separate jobs. MSan is Clang-only and requires all dependencies instrumented.

## Details

Job 1: RUSTFLAGS="-Zsanitizer=address" cargo +nightly nextest run. Job 2: RUSTFLAGS="-Zsanitizer=thread" cargo +nightly nextest run. Use -Zbuild-std for instrumented stdlib. Set ASAN_OPTIONS=detect_leaks=1, UBSAN_OPTIONS=print_stacktrace=1. Add Miri job: cargo +nightly miri test.

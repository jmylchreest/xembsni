# rust-ci-caching

**Decision:** Use Swatinem/rust-cache for Cargo and target directory caching

## Rationale

Swatinem/rust-cache automatically caches ~/.cargo and target/ directories, keyed on Cargo.lock, toolchain, and job. It sets CARGO_INCREMENTAL=0 for reproducible CI builds. Saves 2-5 minutes per job.

## Details

Place after toolchain, before build. Options: shared-key (cross-job sharing), cache-on-failure: true, save-if (conditional). Auto-detects multiple Cargo.lock files. 10 GiB repo cache limit.

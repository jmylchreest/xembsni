# rust-ci-workflow

**Decision:** Separate CI jobs for formatting, linting, testing, and security scanning with dtolnay/rust-toolchain

## Rationale

Separate jobs provide fast feedback on the specific failure type. dtolnay/rust-toolchain is the maintained replacement for the deprecated actions-rs toolchain. Parallel jobs reduce total CI time.

## Details

Jobs: fmt (cargo fmt --all -- --check), clippy (--workspace --all-targets --all-features, RUSTFLAGS="-Dwarnings"), test (cargo nextest run --workspace), deny (cargo deny check), coverage (cargo llvm-cov, main only). Actions: dtolnay/rust-toolchain@stable (@nightly, @1.XX for MSRV). Swatinem/rust-cache@v2 after toolchain setup.

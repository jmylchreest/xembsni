# rust-ci-security

**Decision:** Run cargo-deny in CI on every PR; cargo audit on weekly schedule

## Rationale

cargo-deny checks advisories, licenses, bans, and sources in one pass. Running on every PR catches issues at introduction time. Weekly cargo audit catches newly disclosed vulnerabilities in existing dependencies.

## Details

EmbarkStudios/cargo-deny-action@v2, matrix on checks: [advisories, bans, licenses, sources], command: check ${{ matrix.checks }}. Requires deny.toml -- start permissive, tighten over time. Weekly cron: cargo audit as safety net.

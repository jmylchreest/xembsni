# rust-formatting

**Decision:** Use rustfmt with default settings; enforce in CI with cargo fmt --check

## Rationale

rustfmt eliminates all formatting debates. Default settings match community convention. Consistent formatting reduces review noise.

## Details

CI: cargo fmt --all -- --check. Minimal rustfmt.toml customization. Use #[rustfmt::skip] for exceptions.
